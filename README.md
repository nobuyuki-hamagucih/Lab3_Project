# Lab3_Project
# Git 操作マニュアル作成 
次のような Git 操作のマニュアルを作成します。 
## 作成するファイル 
* Markdown 形式のファイル 
* メインのファイル名は gitmanual.md 
* gitmaual.md に各ファイルへのリンクを作成 
## ファイルの内容 
* git のローカル リポジトリ操作の内容を一通り解説します。 
* git の作業ごとに順番付き箇条書きにしてまとめます。 
* 作業の内容は、sectionX.md というファイル名の別の Markdown ファイルに記述します。X には
箇条書きの番号を入れます。gitmanual.mdにはそのファイルへのリンクを記述します。 
* gitmanual.md、作業ごとの Markdown ファイルは最大でも 500 文字とし、内容を分かりやすく簡
潔にまとめます。 
* README.md ファイルを作成し、gitmanual.md についての簡単な説明を記述します。

# Git 操作 
次のような一連の Git 操作を実行するための作業リストを作成します。 
* git の初期化して、main ブランチを作成する。 
* README.md ファイルを main ブランチにコミットして初期コミットを作成する。 
* work という名前のブランチを main ブランチから作成する。 
* 残りの全てのファイルを work ブランチにコミットする。

# 作業リストの保存 
次のように作業リストを保存します。 
* docs フォルダーを作成する。 
* 作成した docs フォルダーに copilot-generated-task-list.md という名前で作業リストの内容を保存す
る。 
* docs フォルダーを git のバージョン管理の対象外にする。

# 作業リストの作業の実行 
docs/copilot-generated-task-list.md の作業リストを次のように実行します。 
* 順番付き箇条書きのステップごとに、開始と終了時に確認を求めながら、逐次実行する。 
* すべてのメッセージは日本語で生成する。
